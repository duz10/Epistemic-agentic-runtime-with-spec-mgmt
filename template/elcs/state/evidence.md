## How to Add Evidence

```
| E1 | observation | Users complained about slow load times | Support tickets | A3, H2 |
| E2 | measurement | Page load p95 = 3.2s | Performance logs | H2 |
```

Evidence types:
- **observation**: Something you saw/heard
- **measurement**: Quantitative data
- **experiment**: Controlled test results
- **user_input**: Direct user feedback
