ü§ù Human-in-the-Loop Agentic Workflows

ELCS assumes agents do the heavy lifting and humans keep direction and risk under control. Autonomy with a steering wheel.

Typical Flow

1. Agent reads state + spec
2. Agent proposes goals (must pass gates)
3. Human approves risk/priority
4. Agent executes tokens
5. Human reviews outputs at checkpoints

Why it works

Agents are fast but can drift. Humans are slow but set direction. ELCS gives both a shared source of truth ‚Äî the files on disk ‚Äî so neither has to guess what the other meant.
